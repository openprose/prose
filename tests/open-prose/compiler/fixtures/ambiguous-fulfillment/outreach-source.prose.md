# Outreach Source

### Goal

Open customer requests are turned into a customer-facing follow-up view.

### Maintains

The `customer_signal` truth.

- For each open request: a customer-facing follow-up recommendation with the
  evidence and context behind it. Material: the recommendation and the
  evidence digest. Immaterial: any `fetched_at` timestamp.

### Continuity

- self-driven: refresh outreach context at least daily.
