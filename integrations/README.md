# Integrations

Runtime-specific adapters that make OpenProse a first-class citizen on different platforms.

| Integration | Platform | Status |
|-------------|----------|--------|
| [openclaw-plugin](./openclaw-plugin/) | [OpenClaw](https://openclaw.ai) | Alpha — `/prose` commands, subagent execution |

Each integration is a standalone package with its own install instructions, tests, and README.
