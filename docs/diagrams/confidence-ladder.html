What gets protected

Output tool declared outputs are validated before acceptance

Required vs opt-in

  • required: compile, plan, graph, package, publish-check
  • required: deterministic runtime and binary smoke
  • required: skipped live smoke report

Why this matters

The ladder catches drift in package quality, runtime behavior, and live Pi interop before it reaches a release.
