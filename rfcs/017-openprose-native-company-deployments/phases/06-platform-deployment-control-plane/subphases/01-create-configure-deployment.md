# 06.1 Create And Configure Deployment

## Build

- Create deployment from a compiled package version.
- Discover package entrypoints through the OSS deployment contract.
- Allow workflow enablement and environment binding.
- Persist trigger proposals but leave them disabled by default.

## Tests

- Create deployment from `@openprose/examples`.
- Create deployment from `@openprose/prose-openprose` in a fixture path or dev
  package.
- Missing environment prevents distributed workflow activation.
- Run API typecheck and targeted tests.
