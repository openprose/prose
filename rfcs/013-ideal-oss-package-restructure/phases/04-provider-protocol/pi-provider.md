# Superseded Vocabulary: Pi Provider

This page is a historical stub. Phase 04.5 originally described Pi as a
provider. The implemented architecture now uses sharper vocabulary.

## Current Reading

- Pi is the OpenProse graph VM.
- The concrete per-node executor is the Pi node runner.
- Model provider, model ID, thinking level, tools, API key, timeout, and
  session persistence are runtime-profile configuration.
- OpenProse owns graph planning, effect gates, run attempts, artifact
  materialization, eval acceptance, and current/latest pointers.
